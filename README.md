# _Pizza Maker_

#### _Select pizza toppings and size, 6/17/2016_

#### By _**Alan Denison**_

## Description

BDD

this program should allow the user to select size of pizza
  input: large
  output large pizza

this program should allow users to create a pizza and select toppings
  input: cheese
  output: pizza with cheese

this program should display a price based on the options selected
  input: large cheese pizza
  output: $19.99

## Setup/Installation Requirements
_Viewable in modern browsers_

## Technologies Used

CSS, Bootstrap, JQuery
